cascader-gui

cascader-gui is a simple GUI wrapped around xoebus' cascader program. It is part of the at-tools suite from LolSoft, of which I am a founding member. My aim in writing cascader-gui was to make something that was easy to use, and non-command line based, allowing anyone to get involved and for the cascader program to grow and spread.

It is written in Ruby, using the ruby-gtk2 library.

inthelabs

inthelabs is a Ruby client/server with a Rails web frontend. It allows you to see who's in the labs at any point, so that you can decide whether to stay in bed and work on your laptop, or head over to AT to meet up/hang out/code/eat/mingle/etc. It may eventually get some facebook connect lovin', but we'll see.

For more info, see the project page.

massprint

massprint is a wrapper for "lpr" to allow you to easily print a whole load of files. Useful if you're trying to print an entire term's notes in one go...

mailer

mailer is a very simple wrapper for the standard Ruby SMTP library, but doesn't need gems to install. Simply "require mailer", and you're ready to go.

ubuntu-bg

ubuntu-bg is a script to create an xml slideshow file from a folder of pictures, for use on Ubuntu Lucid Lynx's new slideshow-background feature.
